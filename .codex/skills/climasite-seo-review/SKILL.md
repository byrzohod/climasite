# ClimaSite SEO Review

Skip for internal/admin-only surfaces. For public pages, check:
- Crawlability, `robots.txt`, sitemap, no accidental `noindex`.
- Unique title and description per route.
- Canonical links and clean URL structure.
- Open Graph and Twitter metadata.
- JSON-LD for Organization, Website, Breadcrumb, Product, FAQ as applicable.
- One H1, logical heading hierarchy.
- Descriptive image alt text and optimized image formats/sizes.
- Product pages expose price, availability, reviews, and breadcrumbs where applicable.
