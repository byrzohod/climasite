# ClimaSite UI/UX

Priorities:
- Accessibility: contrast, focus, keyboard, labels, accessible names.
- Touch and interaction: 44x44 targets, loading states, clear error feedback.
- Performance: optimized images, reduced motion support, reserve space for async content.
- Responsive layout: no horizontal scroll, stable dimensions, sensible breakpoints.
- Typography and color: readable sizes, good line-height, theme variables only.
- Animation: use transform/opacity, 150-300ms for microinteractions, respect reduced motion.

For operational/e-commerce surfaces, prioritize dense, scannable, task-focused UI over decorative marketing layouts.
